DigitalPulse — Technology & Digital Growth Trends
A data storytelling website that visualises global technology trends through interactive charts, scroll-based narrative sections, and a techy dark/light theme toggle.

Open index.html directly in your browser — no server required.
Everything is self-contained in a single file.

Sections & Data
01 — Global Internet Adoption
Chart: Line / Area · Source: ITU · Our World in Data
02 — Social Media Empires
Chart: Bar (Vertical / Horizontal) · Source: Statista · Platform Reports 2024
03 — E-Commerce Revolution
Chart: Line / Bar (Growth %) · Source: eMarketer · Statista 2024
04 — Code of the Future
Chart: Horizontal Bar · Source: TIOBE Index · Stack Overflow 2024
05 — Key Insights
Format: Cards · Summarised from all four datasets

Features
Dark / Light theme toggle — both themes are techy; preference saved to localStorage
Typewriter effect on hero title — "DIGITAL" types and deletes in a loop
8 chart view toggles — switch between chart styles per section
Scroll reveal animations — sections and cards animate in as you scroll
Hero counters — stats count up on load with eased animation
Floating particles — ambient background animation in hero
Animated background grid — infinite drift effect behind the hero
Platform bar animations — social media bars fill on scroll into view
Responsive — works on desktop, tablet, and mobile
Zero dependencies — only Chart.js via CDN, no build step needed
